# node-front-template

基于 node.js 的前端模板,用到了 es6,async,scss,babel,eslint,gulp,jsx,react,koa2,art-template

[English](README.md) | 中文

## clone

```
$ git clone https://github.com/greedlab/node-front-template.git
```

## 开发环境

```
$ npm install
$ gulp build
$ npm run develop
```

## 生产环境

```
$ npm install --production
$ npm run release
```
