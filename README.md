# node-front-template

template for front end based on node.js

English | [中文](README-CN.md)

## clone

```
$ git clone https://github.com/greedlab/node-front-template.git
```

## develop env

```
$ npm install
$ gulp build
$ gulp watch
$ npm run develop
```

## production env

```
$ npm install
$ gulp release
$ npm run release
```
